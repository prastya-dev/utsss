Tentang Saya

Halo! Nama saya Prastya Ardhi Ramadhan, mahasiswa aktif program studi Informatika PJJ S1 di Universitas Siber Asia. Saya memiliki minat pada bidang web development, server, networking, dan cloud computing. Saya terbiasa belajar dan mengembangkan kemampuan melalui pengalaman langsung, mulai dari pengelolaan Linux Server, Docker, pengembangan aplikasi web, hingga eksplorasi teknologi IoT dan infrastruktur digital.

Saat ini saya juga bekerja sebagai Quality Control di PT Graha Seribu Satu Jaya, yang membantu saya membangun ketelitian, kemampuan analisis, dan penyelesaian masalah. Selain itu, saya menjalankan aktivitas freelance sebagai VPS Reseller dengan pengalaman dalam pengelolaan layanan server dan deployment. Saya percaya bahwa dengan terus belajar dan berkembang secara konsisten, kemampuan teknis dan profesional dapat menjadi fondasi untuk menjadi pribadi yang adaptif, kompeten, dan siap menghadapi perkembangan teknologi.
